# Student Marks Calculator

A simple Python-based tool to help students and teachers calculate total marks, percentages, and letter grades.

## 🚀 Features
- Input custom student names and number of subjects.
- Validates marks and calculates percentages.
- Automatically assigns grades (A-F) based on university standards.
- Clean, formatted console output.
